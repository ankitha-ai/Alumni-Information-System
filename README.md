# Alumni Information System

The Alumni Information System is a web-based application developed to strengthen communication between alumni and educational institutions. It enables alumni to register, update their profiles, participate in events, and stay connected with the institution.

The system provides secure user authentication, role-based access control, event management, profile management, and automated email notifications. Developed using Java (JSP, Servlets), MySQL, Bootstrap, and the MVC architecture, the application ensures scalability, security, and maintainability.
